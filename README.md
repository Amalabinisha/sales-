# SALES PERFORMANCE DASHBOARD ANALYSIS – 2023

## 1. Introduction

Business Intelligence and data visualization play an important role in analyzing sales performance and supporting business decision-making. Dashboards help organizations monitor sales trends, customer behavior, product performance, and regional sales activities in a clear and interactive way.

In this project, a Sales Performance Dashboard was created using Power BI to analyze sales data for the year 2023. The dashboard provides insights into sales amount, quantity sold, product categories, regions, sales representatives, and sales channels.

The project helps businesses identify high-performing products, top sales regions, and sales trends for better strategic planning.

---

# 2. Objective

The main objectives of this project are:

* To analyze sales performance data
* To visualize sales trends and regional performance
* To monitor product category sales
* To compare sales representatives and regions
* To create an interactive Power BI dashboard
* To generate business insights for decision-making

---

# 3. Dataset Description

The dataset used in this project contains sales transaction details for different products and regions.

## Features Used

| Column Name          | Description                        |
| -------------------- | ---------------------------------- |
| Product_ID           | Unique product identifier          |
| Sale_Date            | Date of sale                       |
| Sales_Rep            | Sales representative               |
| Region               | Sales region                       |
| Sales_Amount         | Total sales amount                 |
| Quantity_Sold        | Quantity sold                      |
| Product_Category     | Product category                   |
| Unit_Cost            | Cost per unit                      |
| Unit_Price           | Price per unit                     |
| Customer_Type        | New or returning customer          |
| Discount             | Discount provided                  |
| Payment_Method       | Payment type                       |
| Sales_Channel        | Online or retail sales             |
| Region_and_Sales_Rep | Combined region and representative |

# 8. Business Insights

The dashboard provides several important business insights:

* North and East regions generated higher sales revenue.

* Online sales channels contributed significantly to total sales.
* Some sales representatives performed better than others.
* Monthly sales trends help identify peak sales periods.

These insights help businesses improve marketing, inventory management, and sales strategies.
